# 简介

这是一个用于日常开发所需的一些Android工具类

## 目前包含

1. LankyLog
   
   具备level过滤、文件行号详情的日志工具类

2. HiddenEntrance
   
   以顺序按键的方式进入隐藏入口的工具类

3. ViewUtil
   
   焦点/鼠标指针交互变化（突出当前焦点View）

4. SharedPreferenceUtil
   
   基础数据保存工具类
